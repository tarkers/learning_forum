所有回傳的5個參數中 ID是收到的ID 再傳回去， introduce是data這個<系英文簡寫>這個collection中 type為department_introduce 這個document中 text這個element 是字串,keywords是 <系英文簡寫>這個collection中 type為department_keywords 的這個document 是JSON, tabs 是 <系英文簡寫>這個collection中 type為department_tabs 的這個document 是JSON,boards 是<系英文簡寫>_boards這個collection裡的所有document 是JSON

pseudocode:

    post "./CB/department/<系英文簡寫>" ,parameter:ID
   get parameter <系英文簡寫> as p_NN
   if p_NN == 'YT'
       render('Page5',{ID:ID})
   else
       tmp1_1 = data.p_NN.find(type == department_introduce)
       tmp1_2 = data.p_NN.find(type == department_keywords)
       tmp1_3 = data.p_NN.find(type == department_tabs)
       tmp2 = data.p_NN_boards.find()
       render('Page6',{ID:ID,introduce:tmp1_1[0]['text'],keywords:tmp1_2[0],tabs:tmp1_3[0],data:tmp2})
